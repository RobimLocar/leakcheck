## PIXEL — PRÉ-REQUISITO OBRIGATÓRIO

Antes de subir a campanha, o Pixel precisa estar funcionando:

**1. Instalar Meta Pixel no leakcheck.app**
- Criar Pixel no Meta Events Manager
- Adicionar o código base em todas as páginas (via `<head>`)
- Ou usar Conversions API (server-side) — mais preciso

**2. Disparar evento CompleteRegistration**
- Quando: usuário conecta o Stripe (completa o onboarding gratuito)
- Evento: `fbq('track', 'CompleteRegistration')`
- Parâmetros recomendados:
  ```javascript
  fbq('track', 'CompleteRegistration', {
    content_name: 'Stripe Connection',
    currency: 'USD',
    value: 0
  });
  ```

**3. Verificar no Meta Events Manager**
- Testar com Meta Pixel Helper (extensão Chrome)
- Confirmar que o evento aparece como "Active" no Events Manager
